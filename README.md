## Workspace in Docker

1. Connect to DevContainer in VScode

    ![Alt text](./images/Screenshot%202023-10-25%20at%2018.32.50.png)


2. Open Terminal

    ```
    bundle
    bundle exec jekyll s
    ```

3. Access to localhost:4000
